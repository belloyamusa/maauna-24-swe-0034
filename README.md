# Local Business Directory (Flask App)

## Description
This web application allows users to view and add local businesses.

## Features
- View all businesses
- Add new businesses
- View recently added businesses

## Technologies
- Python
- Flask
- Object-Oriented Programming
- Stack Data Structure
- Datetime API

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

python app.py

3. Open browser and go to

http://127.0.0.1:5000
